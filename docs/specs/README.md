# Technical Specifications

This directory contains detailed technical specifications, standards, and system reference documentation.

## Contents

### System Specifications
- **EFFECTS.md** - Effect system specification (damage types, conditions, durations, etc.)
- **TIMED_EVENTS.md** - Timed event system documentation (scheduling, triggers, execution)
- **ERROR_HANDLING.md** - Error handling standards and conventions

### Language & Syntax
- **MACHINE_TEXT_SYNTAX.md** - Command syntax specification for the machine text language

## Purpose

These documents define:
- Technical standards and conventions
- System behavior specifications
- Syntax and language definitions
- Implementation requirements

## When to Reference

**Implementing effects?** → See [EFFECTS.md](./EFFECTS.md)

**Working with scheduled events?** → Check [TIMED_EVENTS.md](./TIMED_EVENTS.md)

**Writing error handling code?** → Follow [ERROR_HANDLING.md](./ERROR_HANDLING.md)

**Parsing or generating machine text?** → Reference [MACHINE_TEXT_SYNTAX.md](./MACHINE_TEXT_SYNTAX.md)
